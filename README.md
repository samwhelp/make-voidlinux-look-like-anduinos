## Howto


### How to download [make-voidlinux-look-like-anduinos.sh](https://github.com/samwhelp/make-voidlinux-look-like-anduinos/blob/main/make-voidlinux-look-like-anduinos.sh) and run [it](https://raw.githubusercontent.com/samwhelp/make-voidlinux-look-like-anduinos/refs/heads/main/make-voidlinux-look-like-anduinos.sh).

> run

``` sh
bash <(curl -L https://raw.githubusercontent.com/samwhelp/make-voidlinux-look-like-anduinos/refs/heads/main/make-voidlinux-look-like-anduinos.sh)
```


> or run

``` sh
bash <(wget -q https://raw.githubusercontent.com/samwhelp/make-voidlinux-look-like-anduinos/refs/heads/main/make-voidlinux-look-like-anduinos.sh -O - )
```


> or run

``` sh
bash -c "$(curl -fsSL https://raw.githubusercontent.com/samwhelp/make-voidlinux-look-like-anduinos/refs/heads/main/make-voidlinux-look-like-anduinos.sh || wget -q https://raw.githubusercontent.com/samwhelp/make-voidlinux-look-like-anduinos/refs/heads/main/make-voidlinux-look-like-anduinos.sh -O - )"
```


> or run

``` sh
bash -c "$(wget -q https://raw.githubusercontent.com/samwhelp/make-voidlinux-look-like-anduinos/refs/heads/main/make-voidlinux-look-like-anduinos.sh -O - || curl -fsSL https://raw.githubusercontent.com/samwhelp/make-voidlinux-look-like-anduinos/refs/heads/main/make-voidlinux-look-like-anduinos.sh)"
```
